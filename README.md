## Instalacion y Ejecucion
Clonado:
```
$ git clone https://github.com/ramaffei/Backend-dev-test-Logica
```
Entorno virtual:
```
$ python3 -m venv .venv
```
Test:
```
$ pip3 install pytest
$ pytest -vv
```
Ejecucion:
```
$ python3 main.py
```
